# Logos para la Hoja Resumen PDF de Rendiciones

Guardar aquí los dos logos (PNG), con estos nombres exactos:

- `logo-gds.png`        → encabezado del PDF (empresa dueña de la caja: GDS INFRAESTRUCTURA SAC)
- `logo-percytech.png`  → pie del PDF ("Elaborado por PercyTech - Solutions")

Recomendado: PNG con **fondo transparente** (sobre todo el de PercyTech, que en la
imagen original viene con fondo azul). Si solo tienes el de fondo azul, igual sirve;
lo acomodo pequeño en el pie.

Se referencian desde el template del PDF con `public_path('images/rendiciones/...')`.
